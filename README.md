## Requirement

* PHP >= 7.3
* laravel >= 7.0
* dcat-admin >= 2.0

## Installation

### Composer installation

```bash
composer require guanguans/dcat-login-captcha --ansi -v
```

### Backend -> Helpers -> Extensions

Enable this extension.
