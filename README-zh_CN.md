## 环境要求

* PHP >= 7.3
* laravel >= 7.0
* dcat-admin >= 2.0

## 安装

### Composer 安装

```bash
composer require guanguans/dcat-login-captcha --ansi -v
```

### 后台 -> 开发工具 -> 扩展列表

启用本扩展。
